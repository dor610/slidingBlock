About Sliding Block Puzzle
A sliding puzzle, sliding block puzzle, or sliding tile puzzle is a combination puzzle that challenges a player to slide (frequently flat) pieces along certain routes (usually on a board) to establish a certain end-configuration. The pieces to be moved may consist of simple shapes, or they may be imprinted with colours, patterns, sections of a larger picture, numbers, or letters.
The oldest type of sliding puzzle is the fifteen puzzle, invented by Noyes Chapman in 1880; Sam Loyd is often wrongly credited with making sliding puzzles popular based on his false claim that he invented the fifteen puzzle.
About This Game
This game is a type of sliding block puzzle game and it is called fifteen puzzle. The only thing you need to do to win in this game is moving each block until those blocks were sorted.
This game was made by using HTML, CSS and Javascript for core functions with a little bit of Lottie for animation. That is all the technology that were used in the front-end side.
In the back-end side, Java Spring Boot was chosen to build the API that is used to process user's score and save these infomation to database, which is managed in MongoDB Atlas.
